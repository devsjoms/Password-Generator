# 🔐 Password Generator

A simple command-line **Password Generator** built with Python. This application generates random passwords using Python's built-in `secrets` module, making the generated passwords suitable for secure use.

## ✨ Features

* 🔒 Generates secure random passwords
* 🔢 User-defined password length
* 📄 Generate multiple passwords at once
* 🛡️ Uses Python's `secrets` module for cryptographically secure randomness
* 💻 Simple and beginner-friendly command-line interface
* 🔁 Option to generate more passwords without restarting the program

---

## 🛠️ Built With

* Python 3
* `string` module
* `secrets` module

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/devsjoms/Password-Generator.git
```

### Navigate to the project directory

```bash
cd Password-Generator
```

### Run the program

```bash
python password_generator.py
```

> **Note:** Replace `password_generator.py` with the actual filename if it is different.

---

## 📖 How to Use

1. Run the Python file.
2. Enter your desired password length.
3. Enter how many passwords you want to generate.
4. The program will instantly display the generated passwords.
5. Choose whether to generate another set or exit the application.
